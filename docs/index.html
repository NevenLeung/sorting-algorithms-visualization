Merge sort


comparison: 0
swap: 0

算法思想

递归的前半部分，将要排序的数组拆分成尽可能少元素的两部分，直到拆分的数组只有一个元素。 递归的后半部分，根据两个拆分数组之间的元素大小关系进行排列，然后合并成一个新的数组，并把这个合并好且有序的数组依次返回。
